## ダウンロード手順

1. [効果音ラボ](https://soundeffect-lab.info/) にアクセス
2. カテゴリから目的の効果音を探す
3. 試聴して気に入ったらダウンロード
4. `public/se/` フォルダに配置
5. ファイル名を上記の推奨名にリネーム（任意）

## 使い方

### セリフに効果音を追加

```typescript
{
  id: 5,
  character: "zundamon",
  text: "これがポイントなのだ！",
  se: {
    src: "point.mp3",  // public/se/point.mp3
    volume: 0.8,       // 音量（0.0〜1.0）
  },
}
```

### よくある使用パターン

```typescript
// ポイント強調
{ text: "ここが重要なのだ！", se: { src: "point.mp3", volume: 0.7 } }

// 驚きのリアクション
{ text: "えっ!?", emotion: "surprised", se: { src: "surprise.mp3" } }

// 正解発表
{ text: "正解なのだ！", emotion: "happy", se: { src: "correct.mp3" } }

// 場面転換
{ text: "次は〇〇について説明するのだ", se: { src: "transition.mp3" } }
```

## 効果音の使いすぎに注意

- 効果音は**ここぞというところ**で使用
- 毎セリフに効果音を入れると騒がしくなる
- 1動画で5〜10個程度が目安
- 重要なポイントや感情の変化で使う

### NG例
```typescript
// 全セリフに効果音 → 騒がしい
{ text: "今日は〇〇を紹介するのだ！", se: { src: "point.mp3" } },
{ text: "これは便利なのだ！", se: { src: "point.mp3" } },
{ text: "使い方は簡単なのだ！", se: { src: "point.mp3" } },
```

### OK例
```typescript
// 重要なポイントだけ
{ text: "今日は〇〇を紹介するのだ！" },
{ text: "これは便利なのだ！" },
{ text: "特にこの機能がすごいのだ！", se: { src: "point.mp3" } },  // ← ここぞ
```

## BGMについて

BGMも効果音ラボから入手できます。

### 設定方法

`config/bgm.yaml` で設定（`npm run sync-script`で反映）：

```yaml
- src: "background.mp3"  # public/bgm/background.mp3
  volume: 0.3             # 音声の邪魔にならないよう控えめに
  loop: true
  fadeIn: 30              # フェードインのフレーム数
  fadeOut: 30             # フェードアウトのフレーム数
```

> **注意**: `src/data/script.ts`の`bgmConfig`を直接編集しても、`sync-script`実行時に上書きされます。必ず`config/bgm.yaml`を使用してください。

### おすすめBGM

| 用途 | 効果音ラボでの名前 | カテゴリ |
|------|-------------------|----------|
| 解説動画 | 「ほのぼの」系 | BGM |
| 紹介動画 | 「楽しい」系 | BGM |
| 技術解説 | 「クール」系 | BGM |
